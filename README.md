# 🎬 CinemaHub

## 📌 Описание

CinemaHub — это сайт кинотеатра, где можно быстро посмотреть афишу, выбрать фильм и купить билет онлайн. Пользователь может узнать информацию о фильмах, посмотреть сеансы и забронировать удобные места. Также есть личный кабинет, где хранятся купленные билеты и избранные фильмы. Сайт делает поход в кино проще и удобнее.

---

## 👥 Для кого этот сайт

- Люди, которые ходят в кино
- Те, кто любит смотреть новинки
- Пользователи, которым удобно покупать билеты онлайн
- Администраторы кинотеатра

---

## 🚀 Основные возможности

- Просмотр фильмов и афиши
- Информация о фильмах (описание, рейтинг)
- Выбор сеансов
- Покупка билетов
- Личный кабинет
- Избранные фильмы
- Поиск
- Админ-панель

---

## 🧩 Страницы сайта

- / — главная (афиша, новинки)
- /auth — вход, регистрация
- /movies — список фильмов
- /movies/:id — страница фильма
- /profile — личный кабинет
- /profile/tickets — мои билеты
- /favorites — избранное
- /search — поиск
- /admin — админка

---

## 💻 Технологии

**Frontend:**

- Vue.js
- Vue Router

👉 Позволяет удобно создавать интерфейс и управлять состоянием

**Backend:**

- Python (FastAPI)

👉 Подходит для быстрого создания API и логики приложения

**База данных:**

- PostgreSQL

👉 Надежная реляционная база данных, хорошо подходит для работы с билетами, пользователями и связями между ними

---

## 🖼️ Как выглядит сайт (описание страниц)

**Главная (/)**

- Шапка с меню
- Баннер с новинками
- Список популярных фильмов  
    👉 Данные загружаются с API

---

**Каталог (/movies)**

- Фильтры
- Список фильмов  
    👉 GET /api/movies

---

**Фильм (/movies/:id)**

- Постер, описание
- Сеансы  
    👉 GET /api/movies/:id

---

**Бронирование (/booking)**

- Схема зала
- Выбор мест  
    👉 данные мест из API

---

**Профиль (/profile)**

- Данные пользователя
- Переход к билетам и избранному

---

## 🗄️ Данные

**User**

- email
- name
- password_hash
- избранное
- key_word

**Movie**

- title
- description
- genre
- duration
- rating
- poster_url
- date

**Sessions**

- movie_id
- start_time
- price

**Ticket**

- user_id
- session_id
- seat_row
- seat_number

**Favorites**

- user_id
- movie_id

---

## 🔌 API

- POST /auth/register — регистрация
- POST /auth/login — вход
- GET /movies — список фильмов
- GET /movies/:id — фильм
- POST /tickets — купить билет
